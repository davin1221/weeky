       <ul>

       </ul>
     </li>
     <li>
       상세페이지
       <ul>
         <li>날짜 이동</li>
         <li>계획 완료/미완료 변경</li>
         <li>계획 전체 삭제</li>
       </ul>
     </li>
     <li>
       에디터(수정/작성)
       <ul>
         <li>선택된 날짜의 데이터 가져오기</li>
         <li>작성된 정보 수정/삭제 및 새로 작성</li>
       </ul>
     </li>
   </ul>
